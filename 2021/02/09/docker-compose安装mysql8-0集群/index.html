mysql8.0主从集群搭建

前提：docker和docker-compose都安装完毕

  • 将对应配置目录放置于根目录下,点击下载

启动主从数据库容器及配置主数据库

  • 登陆mysql，密码root

  • 新建一个主从复制用的用户并赋权

配置从数据库

  • 登陆mysql，密码root

  • 根据实际的主节点IP，用户名，密码，及日志文件名和位置参数（这两个参数按master配置获取）

设置服务器ID，要求master和slave要唯一；


  • 验证主从集群是否搭建成功，请参考下载压缩包内的《搭建msyql集群操作手册.xlsx》
